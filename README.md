Desenvolvedor backend autodidata focado em Java e Spring Boot. Construo APIs REST completas — com autenticação, testes, migrations e deploy — e integro modelos de IA em fluxos reais de aplicação.

## Projetos

### StudyMind
Plataforma de estudos com onboarding via IA. O usuário conversa com um assistente que coleta seu perfil e gera automaticamente um plano de estudos personalizado de 6 semanas, populando matérias, tópicos e tarefas no banco.

**Backend:** https://studymind-l3ej.onrender.com  
**Frontend:** https://studymind-mu.vercel.app

`Java 17` `Spring Boot 3` `PostgreSQL` `Flyway` `Spring Security` `JWT` `Docker` `Anthropic API`

---

### Biblioteca API
API REST de gerenciamento de biblioteca com autenticação JWT, controle de roles (ADMIN/USER), testes unitários e frontend integrado.

**Backend:** https://biblioteca-api-nqwi.onrender.com  
**Frontend:** https://eduardo-ciudad.github.io/frontend-biblioteca-api/

`Java 17` `Spring Boot 3` `PostgreSQL` `Flyway` `Spring Security` `JWT` `JUnit 5` `Mockito`
